ukl scroll up is an awesome, super lightweight plugin for your wordpress website. 

== Description ==

This plugin will enable your wordpress theme. You can see the scroll up image every page in the bottom right side corner. 



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress



= How do I install this plugin? =

You can install as others regular wordpress plugin. No different way. Please see on installation tab.
